# Dashboard POE AMM Guatemala

Dashboard público y estático del Precio de Oportunidad de la Energía (POE) en el nodo de referencia Guatemala Sur 230 kV.

## Cobertura

- Enero de 2015 a junio de 2026
- 100,776 registros horarios
- Sin horas faltantes ni duplicadas
- Unidad: USD/MWh

## Fuentes

- Archivos anuales de Precios de Oportunidad de la Energía del AMM, 2015–2025
- Hoja PR01 de los Informes de Transacciones Económicas, enero–junio de 2026

Este proyecto no es una publicación oficial del AMM.
